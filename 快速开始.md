## 🚀 三种启动方式（任选其一）

### 方式一：Python脚本启动（推荐）
```bash
# 进入项目目录
cd /home/workspace/可视化标注平台项目

# 安装依赖（首次使用）
pip install -r requirements.txt

# 启动平台
python run_platform.py

# 或指定数据文件
python run_platform.py --data 你的数据.csv
```

### 方式二：一键脚本启动
```bash
# 给予执行权限（首次使用）
chmod +x 一键启动标注平台.sh

# 运行脚本
./一键启动标注平台.sh
```

### 方式三：直接运行主程序
```bash
# 安装依赖
pip install -r requirements.txt

# 直接运行
streamlit run src/visual_annotation_platform.py
```

## 📊 数据文件格式要求

输入CSV文件**必须包含**以下列（列名必须完全一致）：
- `ord.id` - 数据唯一ID
- `muri` - 图片URL（比对图片）
- `uri` - 图片URL（可能存在的备用链接）
- `ord.page` - 网页URL（需要检查的页面）

**示例数据格式**：
```csv
ord.id,muri,uri,ord.page
1,https://example.com/image1.jpg,https://example.com/image1_alt.jpg,https://news.example.com/article1
2,https://example.com/image2.jpg,https://example.com/image2_alt.jpg,https://news.example.com/article2
```

## 🎯 核心功能特性

### 1. **批量图片加载**
- 同时加载所有数据的muri和uri图片
- 支持多线程并发下载
- 自动缓存已下载的图片
- 断点续传支持

### 2. **可视化标注界面**
- 左侧：图片显示区域（muri和uri并排显示）
- 右侧：标注选项区域
- 底部：数据统计和进度显示
- 顶部：导航和操作按钮

### 3. **标注选项系统**
**是否有效（单选）**：
- ✅ 有效数据
- 🎬 电影宣传图
- 🚫 无效数据
- ❌ 不一样的图
- 🏷️ 有水印

**有水印（多选）**：
- 🔹 视觉中国
- 🔹 新华网
- 🔹 中新社
- 🔹 图虫
- 🔹 人民日报
- 🔹 央视总台
- 🔹 其他水印

### 4. **批量处理模式**
- 支持加载所有数据一次性标注
- 实时保存进度
- 支持导出和导入标注结果
- 自动生成统计报告

## 🔧 配置说明

### 主要配置项（config.json）
```json
{
    "data_source": {
        "muri_column": "muri",           // muri列名
        "uri_column": "uri",             // uri列名
        "page_column": "ord.page",       // 网页列名
        "id_column": "ord.id"            // ID列名
    },
    "annotation_options": {              // 标注选项
        "是否有效": ["有效数据", "电影宣传图", "有水印", "无效数据", "不一样的图"],
        "有水印": ["视觉中国", "新华网", "中新社", "图虫", "人民日报", "央视总台", "其他水印"]
    },
    "ui_settings": {                     // 界面设置
        "images_per_page": 20,           // 每页显示图片数
        "image_size": 300,               // 图片显示大小
        "auto_load_webpage": false,      // 是否自动加载网页
        "enable_batch_mode": true        // 是否启用批量模式
    }
}
```

## 📈 输出结果

### 1. **标注结果文件** (outputs/标注结果.csv)
包含原始数据列和以下新增列：
- `是否有效` - 有效性标注结果
- `有水印` - 水印类型（多个用逗号分隔）
- `标注时间` - 标注完成时间
- `标注状态` - 标注状态（已标注/待标注）

### 2. **统计报告** (reports/标注统计报告.json)
- 总数据量统计
- 各类标注分布
- 处理时间统计
- 水印类型分布

### 3. **日志文件** (logs/platform_YYYYMMDD.log)
- 详细的操作日志
- 错误和警告信息
- 性能统计信息

## 🛠️ 常见问题

### Q1：依赖安装失败
```bash
# 使用国内镜像源
pip install -i https://pypi.tuna.tsinghua.edu.cn/simple -r requirements.txt
```

### Q2：图片加载慢
- 检查网络连接
- 减少`images_per_page`数量
- 启用图片缓存功能

### Q3：标注结果未保存
- 检查`outputs`目录权限
- 确保有足够的磁盘空间
- 查看日志文件获取详细错误信息

### Q4：网页无法打开
- 检查端口是否被占用
- 确保防火墙允许相应端口
- 尝试使用其他端口：`--port 8502`

## 📱 工作流程建议

### 1. **数据准备阶段**
- 确保CSV格式正确
- 检查图片URL有效性
- 备份原始数据

### 2. **标注工作阶段**
- 按页面分批标注
- 定期保存进度（每20-30条）
- 疑难数据标记待复核

### 3. **结果验证阶段**
- 导出标注结果
- 抽样检查标注质量
- 生成统计报告

### 4. **数据交付阶段**
- 整理最终结果文件
- 提交统计报告
- 归档日志文件
